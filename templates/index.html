Artificial Neural Network

Artificial neural network is massive parallel Distributed system which has natural propensity to store experimental Knowledge and making them available for use.

Convolutional Neural Network

convolutional neural network is a class of artificial neural network, most commonly applied to analyze visual imagery. A convolutional neural network consists convolution, RELU, pooling and flatten operation.

Recurrent Neural Network

Recurrent Neural Network(RNN) are a type of Neural Network where the output from previous step are fed as input to the current step. The main and most important feature of RNN is Hidden state, which remembers some information about a sequence..

Object Detection

Object recognition is refers to a collection of related tasks for identifying objects in digital photographs. Region-Based Convolutional Neural Networks, or R-CNNs, are a family of techniques for addressing object localization and recognition tasks.

Video Processing

video processing is a particular case of signal processing, in particular image processing, which often employs video filters and where the input and output signals are video files or video streams.
